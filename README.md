Set up Guide

1. make sure you have python installed
2. clone repo
3. create virtual env , steps:-
                        pip install virtualenv
                        virtualenv env
                        .\env\Scripts\activate
 4. intall these modules
    pip install flask flask_sqlalchemy bcrypt email_validator
 5. run command
     python app.py
